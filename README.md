# 🌍 Meu Primeiro Site  

Este é o meu primeiro site criado com **HTML, CSS e JavaScript** no **VS Code**. 🚀  

## 📌 Sobre o Projeto  
O site contém:

## 🛠 Tecnologias Utilizadas
